# 目录分析

- `/arch` 架构相关
  - `/arch/arm/boot/zImage` 编译出的内核镜像
  - `/arch/arm/boot/dts` 设备树
- `/block` 块设备相关
- `/crypto` 各种加密算法
- `/Documentation` 各种文档
- `/driver` 驱动
- `/firmware` 有些外设或模块需要
- `/fs` 文件系统相关
- `/include` 头文件
- `/init` 初始化相关
- `/ipc` 进程间通信
- `/kernel` 内核相关
- `/lib` 库
- `/mm` 内存管理相关算法、机制实现
- `/net` 网络协议
- `/sample` 简单的示例代码
- `/script` 各种脚本
- `/security` 安全相关
- `/sound` 音频相关的，包含驱动
- `/usr` 
- `/virt` 虚拟化相关
- `Kconfig` menuconfig 用到的
- `Makefile` 顶层的 
- `vmlinux` 


做嵌入式linux 开发，关注 boot 里的东西。还有参考现成驱动 driver 里的东西。

老的 uboot 用 uImage 镜像。
